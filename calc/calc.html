formula

ATK = [ (ATK character + ATK wepon) * (1 + ATK Bounus%) ] + ATK flat

  • ATK is the character's final attack value, which can be viewed from the character's attributes screen.
  • ATK Character is the flat attack value the character receives from leveling up and ascensions.
  • ATK Weapon is the weapon's flat attack value.
  • ATK Bonus% is the percentage attack bonus that is obtained from all sources. It is specifically listed as ATK % and not DMG %. This includes buffs from artifacts, weapon passives, skills, etc.
  • ATK Flat is the flat attack value obtained from from artifacts, skills, food, etc.

outDMG = ATK * Ability% * (1+DMG Bounus%)

  • DMG Bonus% is the sum of all damage bonuses applicable to the attack.
    • These bonuses will always be referred to in-game using the "DMG" keyword (e.g., Elemental/Physical/Burst/Skill/Normal Attack/Charged Attack/All DMG%, increased DMG against enemies affected by Electro/Pyro/Hydro/Cryo).
    • Notably, ATK and ATK% bonuses are NOT DMG bonuses.

outDMGcrit = outDMG * (1+critDMG%)

inDMG = outDMG * DEF Multi * RES Multi

DEF Multi = (Lvl char + 100) / (Lvl char + Lvl enemy + 200)

RES Multi = 1 - RES // where 0 < res < 0.75

RES = RES base + RES bounus + RES debuff
