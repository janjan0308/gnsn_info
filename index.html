    聖遺物SubStats

    Along with the main stat, artifacts can have up to 4 sub-stats. Every 4 levels, an artifact gains a new sub-stat if it has less than 4, or upgrades an existing one if it already has 4. An artifact's sub-stat cannot be the same as the main stat.
